# Team-Up

## Description
* Team-Up is a gaming social networking app, where you can follow your favorite games, and add users with similar game interest to team up!

## Technologies Used
* `MERN (MongoDB, Express.js, React, and Node.js)`
* `GraphQl`
* `JSON Web Token`
